📌 Sobre o Projeto

O Mapa de Controle de Salas é uma aplicação web desenvolvida com foco educacional, criada para organizar e visualizar a ocupação de salas em uma escola, empresa ou instituição.

O sistema permite identificar quais salas estão:

✅ Livres

🔴 Em uso

🟡 Em manutenção

O objetivo principal é evitar conflitos de agendamento e facilitar a organização dos ambientes físicos.

🎯 Objetivo do Sistema

Este projeto foi desenvolvido para:

Simular um sistema real de organização de ambientes;

Aplicar conceitos de HTML, CSS e responsividade;

Servir como base para futuras implementações com JavaScript, PHP e banco de dados;

Demonstrar estruturação e organização de interface web;

Publicar e versionar um projeto real utilizando Git e GitHub.

🏫 Aplicação Prática

O sistema pode ser utilizado em:

Escolas (controle de laboratórios e salas de aula)

Escritórios (salas de reunião)

Instituições públicas

Centros de treinamento
